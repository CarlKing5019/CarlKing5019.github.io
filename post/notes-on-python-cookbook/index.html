Notes on Python Cookbook

Posted on 2017-07-08 | In Coding |
A note on Python Cookbook.

《Python Cookbook, 3rd Edition》, 中文名称:《Python Cookbook, 第三版》.
英文版有 706 页, 中文版有 619 页, 出版于 May 2013, 涵盖 15 个方面，共计 260 个例子. 本书介绍了Python应用在各个领域中的一些使用技巧和方法, 覆盖了Python应用中的很多常见问题, 并提出了通用的解决方案, 适合有一定Python基础的用户阅读.
这本书的中文版有人民邮电出版社 2015 年出版的 《Python Cookbook, (第 3 版)中文版》, 翻看了网上一个 99 页的样书 (对, 就是微盘下载的那个大小为 30M 的, 其实只包含了前两张的内容, 而且还是图片版的), 发现翻译的有诸多错误, 此外对于代码也没有高亮处理, 使用的也不是等宽字体. 另一个中文版是 readthedocs 上熊能翻译的版本, 总体翻译质量较高, 但也有少许错误, 方便的是既有在线版本的, 也有 pdf 版本的. 对不少量翻译错误可以对照着英文版阅读.
相关链接: 中文版 pdf, 中文版在线阅读, 中文版翻译项目主页, 英文版 pdf, O Reilly website, Errata
